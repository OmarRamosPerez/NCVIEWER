# NCViewer
[![Static Badge](https://img.shields.io/badge/pypi-3.4.4-brightgreen)](https://pypi.org/project/ncviewer/)

NetCDF file viewer inspired by ncview, built with PyQt5 and Matplotlib.
<br><br>

# Version
1.0.2
<br><br>


## Features

- Browse NetCDF variables and dimensions.
- 2D and 1D plotting with colormap selection.
- Navigate time/levels with arrow controls.
- Displays time as a formatted date when available.
- Geographic coordinates (lon/lat) on axes.

## Installation
You can install **GOES** on Python on Linux, Windows, or other operating systems using the following command.


**Using pip** (recommended):

``` 
pip install ncviewer
```

Check if the package was installed with:

```
pip show ncviewer
```
